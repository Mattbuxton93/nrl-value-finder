How It Works

Value = Edge
Edge % = (True Probability × Decimal Odds − 1) × 100

Consensus Model
For H2H, the app uses the average of all bookmaker odds as the "true" probability baseline. Where one bookie is higher than the consensus, that's a value opportunity.

Kelly Criterion
f = (b×p − q) / b — use Quarter Kelly (×0.25) to reduce variance.
